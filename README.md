# Full-Stack Vacation Viewing Website

A web page to view and manage vacations, either as a user and an admin.

### Tech-stack:
Front-end: React + TypeScript + Vite + Material UI

Back-end: Node + TypeScript + OpenAI API

Data: MySQL Server
